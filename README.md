# CodeWinOptimizer

Windows Gaming Optimization Tool. 
Developed with [Wails](https://wails.io/), providing a desktop interface using web technologies and Go backend.

## 🚀 Requisitos previos

Para poder construir y ejecutar este proyecto necesitas:

- [Go](https://go.dev/doc/install) 1.20 o superior.
- [Wails](https://wails.io/docs/gettingstarted/installation).

## 🛠️ Desarrollo en vivo (Live Development)

Para ejecutar en modo de desarrollo con recarga automática (live reload):

```bash
wails dev
```

## 📦 Construcción (Build)

Para generar el ejecutable de la aplicación para Windows:

```bash
wails build
```

El ejecutable se generará en la carpeta `build/bin/`.
